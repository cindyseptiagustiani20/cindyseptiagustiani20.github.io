3. Array

Pengertian Array

Array adalah tipe data yang digunakan untuk menyimpan kumpulan data dengan tipe data yang sama.


Array/Larik merupakan Struktur Data Sederhana yang dapat didefinisikan sebagai pemesanan alokasi memory sementara pada komputer.Array dapat didefinisikan sebagai suatu himpunan hingga elemen yang terurut dan homogen


Terurut

Dapat diartikan bahwa elemen tersebut dapat diidentifikasi sebagai elemen pertama, elemen kedua dan seterusnya sampai elemen ke-n.


Homogen

Adalah bahwa setiap elemen dari sebuah Array tertentu haruslah mempunyai type data yang sama.



Sebuah Array dapat mempunyai elemen yang seluruhnya berupa integer atau character atau String bahkan dapat pula terjadi suatu Array mempunyai elemen berupa Array.

Karakteristik Array

Array memiliki beberapa karakteristik, yaitu:

Array adalah tipe data yang terurut. Artinya, setiap elemen array memiliki posisi yang berbeda-beda. Elemen pertama disebut dengan indeks 0, elemen kedua disebut dengan indeks 1, dan seterusnya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#terurut

Array adalah tipe data yang homogen. Artinya, setiap elemen array memiliki tipe data yang sama. Tipe data yang digunakan dapat berupa tipe data primitif seperti int, float, double, char, dan sebagainya. Tipe data yang digunakan juga dapat berupa tipe data kompleks seperti string, array, pointer, structure, dan sebagainya.

Referensi: https://www.petanikode.com/c-dasar/tipe -data/#homogen

Array adalah tipe data yang statis. Artinya, ukuran array harus dideklarasikan terlebih dahulu sebelum digunakan. Ukuran array tidak dapat diubah setelah dideklarasikan.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#statis

Array adalah tipe data yang dinamis. Artinya, ukuran array dapat diubah setelah dideklarasikan. Ukuran array dapat diubah dengan menggunakan fungsi realloc().

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#dinamis

Array adalah tipe data yang terstruktur. Artinya, array dapat menyimpan data dengan tipe data yang berbeda-beda. Data yang disimpan dapat berupa data primitif seperti int, float, double, char, dan sebagainya. Data yang disimpan juga dapat berupa data kompleks seperti string, array, pointer, structure, dan sebagainya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#terstruktur

Array adalah tipe data yang abstrak. Artinya, array tidak dapat diakses secara langsung oleh programmer. Array hanya dapat diakses melalui indeksnya saja.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#abstrak

Array Satu Dimensi

Array satu dimensi adalah array yang memiliki satu indeks. Array satu dimensi juga disebut sebagai array linear.

int jumlah_elemen = sizeof(nama_array) / sizeof(tipe_data);

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#array-satu-dimensi

Array satu dimensi dapat digambarkan sebagai berikut:

Rumus : @A[i] = B + (i – 1) * L

A = Nama Array

i = Indeks Array

B = Basis Array

L = Ukuran Elemen Array

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#array-satu-dimensi

Contoh array satu dimensi adalah sebagai berikut:

int angka[5];

char huruf[5];

float bilangan[5];

double bilangan[5];

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#array-satu-dimensi

Array Dua Dimensi

Array dua dimensi adalah array yang memiliki dua indeks. Array dua dimensi juga disebut sebagai array matriks.

int jumlah_elemen = sizeof(nama_array) / sizeof(tipe_data);

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#array-dua-dimensi

Array dua dimensi dapat digambarkan sebagai berikut:

Rumus : @A[i][j] = B + (i – 1) * L + (j – 1) * C

A = Nama Array

i = Indeks Array Pertama

j = Indeks Array Kedua

B = Basis Array

L = Ukuran Elemen Array Pertama

C = Ukuran Elemen Array Kedua

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#array-dua-dimensi

Contoh array dua dimensi adalah sebagai berikut:

int angka[5][5];

char huruf[5][5];

float bilangan[5][5];

double bilangan[5][5];

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#array-dua-dimensi

Array Tiga Dimensi

Array tiga dimensi adalah array yang memiliki tiga indeks. Array tiga dimensi juga disebut sebagai array kubus.

int jumlah_elemen = sizeof(nama_array) / sizeof(tipe_data);

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#array-tiga-dimensi

Array tiga dimensi dapat digambarkan sebagai berikut:

Rumus : @A[i][j][k] = B + (i – 1) * L + (j – 1) * C + (k – 1) * D

A = Nama Array

i = Indeks Array Pertama

j = Indeks Array Kedua

k = Indeks Array Ketiga

B = Basis Array

L = Ukuran Elemen Array Pertama

C = Ukuran Elemen Array Kedua

D = Ukuran Elemen Array Ketiga

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#array-tiga-dimensi

Contoh array tiga dimensi adalah sebagai berikut:

int angka[5][5][5];

char huruf[5][5][5];

float bilangan[5][5][5];

double bilangan[5][5][5];

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#array-tiga-dimensi
