Struktur Data

2. Tipe Data

Pengertian Tipe Data

Tipe data adalah jenis data yang tersimpan pada variabel. Tipe data menentukan ukuran dan jenis data yang dapat disimpan pada variabel. Tipe data juga menentukan operasi apa saja yang dapat dilakukan pada variabel tersebut.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/

Jenis-jenis Tipe Data

Berikut merupakan beberapa jenis tipe data

Tipe data boolean adalah tipe data yang hanya memiliki dua nilai, yaitu true dan false. Nilai true dapat diartikan sebagai 1, sedangkan false dapat diartikan sebagai 0.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#boolean

Tipe data character adalah tipe data yang digunakan untuk menyimpan karakter. Karakter yang disimpan dapat berupa huruf, angka, simbol, dan karakter khusus lainnya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#character

Tipe data byte adalah tipe data yang digunakan untuk menyimpan bilangan bulat yang memiliki rentang nilai yang sama dengan tipe data char. Tipe data byte memiliki ukuran 1 byte atau 8 bit.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#byte

Tipe data short adalah tipe data yang digunakan untuk menyimpan bilangan bulat yang memiliki rentang nilai yang lebih besar dari tipe data byte. Tipe data short memiliki ukuran 2 byte atau 16 bit.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#short

Tipe data int adalah tipe data yang digunakan untuk menyimpan bilangan bulat yang memiliki rentang nilai yang lebih besar dari tipe data short. Tipe data int memiliki ukuran 4 byte atau 32 bit.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#int

Tipe data long adalah tipe data yang digunakan untuk menyimpan bilangan bulat yang memiliki rentang nilai yang lebih besar dari tipe data int. Tipe data long memiliki ukuran 8 byte atau 64 bit.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#long

Tipe data float adalah tipe data yang digunakan untuk menyimpan bilangan pecahan yang memiliki rentang nilai yang lebih besar dari tipe data long. Tipe data float memiliki ukuran 4 byte atau 32 bit.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#float

Tipe data double adalah tipe data yang digunakan untuk menyimpan bilangan pecahan yang memiliki rentang nilai yang lebih besar dari tipe data float. Tipe data double memiliki ukuran 8 byte atau 64 bit.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#double

Tipe data string adalah tipe data yang digunakan untuk menyimpan kumpulan karakter. Karakter yang disimpan dapat berupa huruf, angka, simbol, dan karakter khusus lainnya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#string

Tipe data array adalah tipe data yang digunakan untuk menyimpan kumpulan data dengan tipe data yang sama. Data yang disimpan dapat berupa data primitif seperti int, float, double, char, dan sebagainya. Data yang disimpan juga dapat berupa data kompleks seperti string, array, pointer, structure, dan sebagainya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#array

Tipe data pointer adalah tipe data yang digunakan untuk menyimpan alamat memori dari variabel lain. Pointer menyimpan alamat memori dari variabel lain, bukan nilai dari variabel tersebut.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#pointer

Tipe data structure adalah tipe data yang digunakan untuk menyimpan kumpulan data dengan tipe data yang berbeda. Data yang disimpan dapat berupa data primitif seperti int, float, double, char, dan sebagainya. Data yang disimpan juga dapat berupa data kompleks seperti string, array, pointer, structure, dan sebagainya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#structure

Tipe data union adalah tipe data yang digunakan untuk menyimpan kumpulan data dengan tipe data yang berbeda. Data yang disimpan dapat berupa data primitif seperti int, float, double, char, dan sebagainya. Data yang disimpan juga dapat berupa data kompleks seperti string, array, pointer, structure, dan sebagainya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#union

Tipe data enumeration adalah tipe data yang digunakan untuk menyimpan kumpulan data dengan tipe data yang berbeda. Data yang disimpan dapat berupa data primitif seperti int, float, double, char, dan sebagainya. Data yang disimpan juga dapat berupa data kompleks seperti string, array, pointer, structure, dan sebagainya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#enumeration

Tipe data file adalah tipe data yang digunakan untuk menyimpan kumpulan data dengan tipe data yang berbeda. Data yang disimpan dapat berupa data primitif seperti int, float, double, char, dan sebagainya. Data yang disimpan juga dapat berupa data kompleks seperti string, array, pointer, structure, dan sebagainya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#file

Tipe Bilangan Real

Tipe data bilangan real adalah tipe data yang digunakan untuk menyimpan bilangan pecahan. Tipe data bilangan real terdiri dari tipe data float dan double.

Tipe data ini akan memiliki keterangan jangkauan, jumlah digit penting (berarti) dan ukuran. Digit berarti ini penting diperhatikan karena ini berhubungan dengan tingkat ketelitian data yang disajikan

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#bilangan-real

Tipe Data Karakter

Tipe data karakter adalah tipe data yang digunakan untuk menyimpan karakter. Karakter yang disimpan dapat berupa huruf, angka, simbol, dan karakter khusus lainnya.

Contoh tipe data karakter adalah sebagai berikut:

char karakter = 'A';

char karakter = '1';

char karakter = '@';

char karakter = ' '; // spasi

char karakter = '\n'; // baris baru

char karakter = '\t'; // tab

char karakter = '\0'; // null

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#tipe-data-karakter

Tipe data string adalah tipe data yang digunakan untuk menyimpan kumpulan karakter. Karakter yang disimpan dapat berupa huruf, angka, simbol, dan karakter khusus lainnya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#string

Tipe data string[x] adalah tipe data yang digunakan untuk menyimpan kumpulan karakter dengan panjang maksimal x. Karakter yang disimpan dapat berupa huruf, angka, simbol, dan karakter khusus lainnya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#string

Tipe Data Boolean

Tipe data boolean adalah tipe data yang hanya memiliki dua nilai, yaitu true dan false. Nilai true dapat diartikan sebagai 1, sedangkan false dapat diartikan sebagai 0.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#boolean

Tipe Data Bentukan

Tipe data bentukan adalah tipe data yang digunakan untuk menyimpan kumpulan data dengan tipe data yang berbeda. Data yang disimpan dapat berupa data primitif seperti int, float, double, char, dan sebagainya. Data yang disimpan juga dapat berupa data kompleks seperti string, array, pointer, structure, dan sebagainya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#tipe-data-bentukan

Macam-macam Tipe Data

Berikut merupakan beberapa macam tipe data

Tipe data primitif adalah tipe data yang sudah tersedia di dalam bahasa pemrograman. Tipe data primitif juga disebut sebagai tipe data bawaan (built-in data type).

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#tipe-data-primitif

Tipe data non-primitif adalah tipe data yang dibuat oleh programmer. Tipe data non-primitif juga disebut sebagai tipe data buatan (user-defined data type).

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#tipe-data-non-primitif

Tipe data statik adalah tipe data yang tidak dapat diubah setelah dideklarasikan. Tipe data statik juga disebut sebagai tipe data konstan (constant data type).

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#tipe-data-statik

Tipe data dinam is adalah tipe data yang dapat diubah setelah dideklarasikan. Tipe data dinamis juga disebut sebagai tipe data variabel (variable data type).

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#tipe-data-dinamis

Tipe data homogen adalah tipe data yang hanya dapat menyimpan satu jenis data saja. Tipe data homogen juga disebut sebagai tipe data sederhana (simple data type).

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#tipe-data-homogen

Tipe data heterogen adalah tipe data yang dapat menyimpan beberapa jenis data sekaligus. Tipe data heterogen juga disebut sebagai tipe data kompleks (complex data type).

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#tipe-data-heterogen

Tipe data terstruktur adalah tipe data yang terdiri dari beberapa tipe data lainnya. Tipe data terstruktur juga disebut sebagai tipe data majemuk (compound data type).

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#tipe-data-terstruktur

Tipe data abstrak adalah tipe data yang tidak dapat diakses secara langsung oleh programmer. Tipe data abstrak juga disebut sebagai tipe data abstrak (abstract data type).

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#tipe-data-abstrak
