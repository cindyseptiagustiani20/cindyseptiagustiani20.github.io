Struktur Data

4. Stack

Pengertian Stack

Stack adalah struktur data yang digunakan untuk menyimpan data dengan prinsip LIFO (Last In First Out). Prinsip ini berarti data yang terakhir masuk akan menjadi data pertama yang keluar.

Stack dapat digambarkan sebagai tumpukan buku. Buku yang terakhir ditumpuk akan menjadi buku yang pertama diambil.

Stack dapat diimplementasikan dengan menggunakan array atau linked list.

Stack memiliki dua operasi utama, yaitu push dan pop. Operasi push digunakan untuk menambahkan data ke dalam stack. Operasi pop digunakan untuk mengambil data dari stack.
