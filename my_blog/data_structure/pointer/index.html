Struktur Data

11. Pointer

Pengertian

Pointer adalah variabel yang menyimpan alamat memori dari variabel lain. Pointer digunakan untuk mengakses variabel lain secara tidak langsung.

Contoh

Contoh penggunaan pointer dalam bahasa pemrograman C++:

                    
                        #include <iostream>
                        using namespace std;

                        int main() {
                            int a = 10;
                            int *p;

                            p = &a;

                            cout << "Nilai dari a: " << a << endl;
                            cout << "Alamat dari a: " << &a << endl;
                            cout << "Nilai dari p: " << p << endl;
                            cout << "Alamat dari p: " << &p << endl;
                            cout << "Nilai yang ditunjuk oleh p: " << *p << endl;

                            return 0;
                        }

Output:

                    
                        Nilai dari a: 10
                        Alamat dari a: 0x61ff04
                        Nilai dari p: 0x61ff04
                        Alamat dari p: 0x61ff00
                        Nilai yang ditunjuk oleh p: 10

Perbedaan Variabel dan Pointer

Perbedaan antara variabel dan pointer adalah:

Variabel Pointer
Menyimpan nilai Menyimpan alamat memori
Memiliki nilai Memiliki alamat memori
Memiliki alamat memori Memiliki alamat memori
Memiliki nilai yang dapat diubah Memiliki alamat memori yang dapat diubah
Memiliki alamat memori yang tetap Memiliki nilai yang tetap

Kelebihan

Kelebihan dari pointer adalah:

  • Memungkinkan untuk mengakses variabel lain secara tidak langsung.
  • Memungkinkan untuk mengakses alamat memori dari variabel lain.

Kekurangan

Kekurangan dari pointer adalah:

  • Memiliki resiko yang tinggi dalam penggunaannya.
  • Memiliki resiko yang tinggi dalam pengalokasian memori.

Kesimpulan

Pointer adalah variabel yang menyimpan alamat memori dari variabel lain. Pointer digunakan untuk mengakses variabel lain secara tidak langsung. Pointer memiliki kelebihan dan kekurangan dalam penggunaannya.
