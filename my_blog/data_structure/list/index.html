Struktur Data

6. List

Pengertian List

List adalah struktur data yang digunakan untuk menyimpan sekumpulan data. List memiliki dua jenis, yaitu singly linked list dan doubly linked list.

List merupakan sebuah pemikiran/konsep struktur data yang sangat dasar pada pemrograman agar lebih fleksibel.

Setiap elemen akan ditambahkan saat dibutuhkan, tidak dialokasikan dengan tempat tertentu dari awal.

Linked List adalah salah satu bentuk struktur data, berisi kumpulan data (node) yang tersusun secara sekuensial, saling sambung-menyambung dan dinamis.

Perbedaan Karakteristik Array dan Linked List

Perbedaan karakteristik array dan linked list adalah sebagai berikut:

Array Linked List
Array adalah struktur data yang menyimpan data secara terindeks. Linked List adalah struktur data yang menyimpan data secara terindeks.
Array memiliki ukuran yang tetap. Linked List memiliki ukuran yang dinamis.
Array memiliki alokasi memori yang statis. Linked List memiliki alokasi memori yang dinamis.
Array memiliki akses yang cepat. Linked List memiliki akses yang lambat.
Array memiliki alokasi memori yang statis. Linked List memiliki alokasi memori yang dinamis.

Konsep Pointer dan Linked List

Pointer adalah variabel yang berisi alamat memori dari variabel lain. Pointer menyimpan alamat dari variabel lain.

Linked List adalah struktur data yang terdiri dari node-node yang saling terhubung. Setiap node memiliki data dan pointer yang menunjuk ke node berikutnya.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#pointer

Perbedaan Perubah Statis Dan Dinamis

Isi Memory pada lokasi tertentu (nilai perubah) adalah data sesungguhnya yang akan diolah.

Nilai perubah adalah alamat lokasi lain yang menyimpan data sesungguhnya. Dengan demikian data yang sesungguhnya dapat dimasukkan secara langsung.

Deklarasi Pointer

Pointer digunakan sebagai penunjuk ke suatu alamat memori

Dalam pemrograman C++, Type Data Pointer dideklarasikan dengan bentuk umum :

type_data *nama_variabel;

Type Data dapat berupa sembarang type data, misalnya char, int atau float. Sedangkan Nama variabel merupakan nama variabel pointer.

Keuntungan List

Keuntungan list adalah sebagai berikut:

  • Penggunaan memori yang dinamik (Kita dapat mengatur penggunaan memori sehingga bisa lebih hemat).
  • Penambahan dan penghapusan elemen lebih mudah.
  • Ukuran list bisa berubah-ubah.
  • Alamat elemen pertama dari suatu list, dapat diacu oleh First(L).
  • Nilai yang dibawanya dapat diacu dengan info(P).

Jenis Link List

Jenis link list adalah sebagai berikut:

Singly Linked List adalah jenis linked list yang paling sederhana. Setiap node menyimpan data dan alamat dari node berikutnya.

Doubly Linked List adalah jenis linked list yang setiap node menyimpan data dan alamat dari node sebelumnya dan node berikutnya.

Circular Linked List adalah jenis linked list yang setiap node menyimpan data dan alamat dari node berikutnya. Pada jenis linked list ini, node terakhir akan menunjuk ke node pertama.

Referensi: https://www.petanikode.com/c-dasar/tipe-data/#jenis-linked-list

Operasi Pada List

Operasi pada list adalah sebagai berikut:

Operasi create adalah operasi untuk membuat list kosong.

Operasi insert adalah operasi untuk menambahkan elemen baru pada list.

Operasi delete adalah operasi untuk menghapus elemen pada list.

Operasi update adalah operasi untuk mengubah elemen pada list.

Operasi read adalah operasi untuk menampilkan elemen pada list.
